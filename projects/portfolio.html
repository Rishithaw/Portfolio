Project Case Study

Portfolio Website

A responsive personal website built to present my background, skills, education, services, projects, and contact information.

Live Demo View Source Back to Projects

Role and Scope

I designed and built this portfolio end-to-end to present projects, technical strengths, and contact options clearly for recruiters and hiring managers.

Challenge

The challenge was making the site visually consistent and professional while keeping content easy to scan on both desktop and mobile screens.

Technical Decisions

  • Built a reusable design system with CSS variables for color, spacing, and typography.
  • Added project detail pages to provide stronger project context than short cards alone.
  • Applied SEO and accessibility improvements across key pages and forms.

Tech Stack

HTML, CSS, JavaScript, responsive design, accessibility improvements, and Vercel deployment.

Outcome Snapshot

  • Published a deployed portfolio with case studies, contact flow, and resume access.
  • Improved project storytelling and visual consistency across all pages.
  • Practiced iterative UX refinement through multiple rounds of feedback.
